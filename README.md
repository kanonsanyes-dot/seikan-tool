# SeikanTool（生管ツール）

中小製造業向けの軽量な生産管理・工程スケジューラーWebツールです。
CSV取込、受注一覧、マスタ管理、集計レポート、工程スケジューラー、進捗表をローカル環境で動かします。

## 起動方法

```bat
cd seikan_tool
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

ブラウザで開く：

```text
http://127.0.0.1:5000
```

## 初回デバッグの順番

1. `python app.py` で起動できるか
2. ダッシュボードが開くか
3. 出荷先マスタ・品名マスタを追加できるか
4. `sample_orders.csv` を受注取込できるか
5. 受注一覧で照合OK / 未登録が表示されるか
6. 集計レポートが表示されるか
7. Excel出力できるか
8. 受注一覧から「工程生成」する
9. スケジューラーでカードが出るか
10. カード移動後に保存されるか
