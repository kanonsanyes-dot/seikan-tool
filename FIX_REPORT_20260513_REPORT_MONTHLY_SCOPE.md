# Fix5: 集計レポート 月別タブのフィルタ仕様明確化

## 背景
Claudeレビューで、`monthly_q`のみ対象月フィルタが外れており、客先別・品名別だけが対象月で絞られる点が確認された。

## 判断
月別タブは会議用の推移確認として「全期間推移」を見る用途があるため、既定値は全期間表示のままとした。
ただし意図が曖昧にならないよう、UIで切り替え可能にした。

## 修正内容
- `build_summary(target_month=None, monthly_scope="all")` に変更
- `monthly_scope="all"` の場合、月別タブは全期間推移
- `monthly_scope="filtered"` の場合、月別タブも対象月で絞る
- 客先別・品名別は従来通り対象月で絞る
- 集計レポート画面に「月別タブの表示範囲」セレクトを追加
- Excel出力にも `monthly_scope` を引き継ぐ

## 確認ポイント
1. 対象月未指定で全タブが表示されること
2. 対象月指定＋全期間推移で、月別は全期間、客先別・品名別は対象月になること
3. 対象月指定＋対象月だけ表示で、月別も対象月のみになること
4. Excel出力で同じ条件が反映されること
